# Linear 对标复刻资料库

本目录用于沉淀 Cruise 对标 Linear Web 的取证资料、结构分析、差距分析与实施路线图。

## 目录说明

- `page-inventory.md`：Linear 页面盘点
- `route-map.md`：路由与导航结构
- `domain-model.md`：领域模型与实体关系
- `api-catalog.md`：接口/请求目录
- `ui-style-guide.md`：视觉风格与设计 token 观察
- `gap-analysis.md`：Cruise 与 Linear 的差距分析
- `implementation-roadmap.md`：复刻实施路线图
- `screenshots/`：截图资料
- `har/`：HAR 或网络请求摘要
- `dom/`：DOM、页面文本、结构快照
- `flows/`：关键交互流程记录

## 约束

- 只在 `docs/` 内沉淀本项目文档
- 不向 `references/` 写入任何内容
- 以“只读采集 Linear、在 Cruise 中复刻体验”为主
